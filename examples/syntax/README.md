# Syntax Examples

These are standalone spell examples demonstrating Glyph notation. They are not part of the dehydration/rehydration experiments.

## Files

| Spell | Demonstrates |
|-------|--------------|
| `tokenize.spell` | Basic spell with obligations |
| `parse.spell` | Composition via dependencies (`>`) |
| `inference_abi.spell` | Multi-entity spell with multiple components |
